# guzeyer.com — Personal Website

My personal website, built with a focus on performance, fluid animations, and pixel-perfect design.

**Live at [guzeyer.com](https://guzeyer.com)**

---

## Stack

| Technology | Purpose |
|---|---|
| React 19 + TypeScript | UI & type safety |
| Vite | Build tooling |
| Tailwind CSS v4 | Styling |
| Framer Motion | Animations & transitions |
| Jotai | Global state |

---

## Features

- **Typewriter effect** — cycling through roles in the hero
- **Scroll-triggered animations** — sections animate in as you scroll
- **Animated skill bars** — progress bars with per-skill colors and glow
- **Neural network visualization** — animated SVG in the AI section
- **Particle field** — canvas-based background animation
- **Scroll progress bar** — top-of-page reading indicator
- **Magnetic buttons** — subtle 3D hover effect on CTAs
- **Lazy-loaded sections** — code-split with React Suspense for fast initial load
- **CSS Modules** — component-scoped styles alongside Tailwind

---

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

---

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `master` triggers an automatic production deployment.
